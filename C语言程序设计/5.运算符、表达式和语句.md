# 1.b=a++和b=++a的区别
```c
int main(void)
{
	int a = 1, b = 1;
	b = a++;
	printf("%d %d", a, b);
	return 0;
}
```
上述代码属于先将把b的值赋值给a，然后a再进行+1的运算；最终会输出**2 1；**
```c
int main(void)
{
	int a = 1, b = 1;
	b = ++a;
	printf("%d %d", a, b);
	return 0;
}
```
上述代码先执行给a+1的操作，然后再将a的值赋值给b，会输出**2 2**；
```c
return n++;
return ++n;
return n + 1;
```
上述代码中，<br />第1行仅会返回n，因为++运算符放在之后，会先返回再++；<br />第2行和第3行都会正确执行想要的操作。

# 2.分数运算
```c
#include <stdio.h>
int main(void)
{
	printf("%f", 4.0/3);
	return 0;
}
```
如果要输出**4/3**的结果的小数，4或3其中一个要写成如**4.0**这种形式，才会输出**1.333333**。将其中一个操作数替换为浮点型来进行计算。

# 3.条件运算符
条件运算符（三目操作符）根据条件选择返回两个值之一。<br />语法如下：
```c
条件表达式 ? 结果1 : 结果2
```
结果1是当条件表达式成立的时候会运行的语句（即条件真），而结果2是当条件表达是不成立时会运行的语句（即条件假）。<br />如：
```c
#include <stdio.h>

int main(void) 
{
    int x = 5;
    int y = (x > 0) ? 10 : -10;

    printf("y = %d\n", y);

    return 0;
}
```
上述代码x > 5成立，运行结果1，即返回一个10的值。最终输出10。

# 4.逗号表达式
逗号表达式每个语句都会执行，但最终返回的是最后一个语句的结果。<br />语法如下：
```c
表达式1, 表达式2, 表达式3, ..., 表达式n
```
代码示例：
```c
#include <stdio.h>

int main(void) {
    int x = 5, y = 10, z;
    
    z = (x++, y++, x + y); // 逗号表达式中的x++和y++会被依次执行，最后的结果是x+y赋给z

    printf("z = %d\n", z);

    return 0;
}

```
最终会输出17。

# 6.位操作运算符
这些运算符与二进制有关。
<a name="O6vKs"></a>
## (1)&
位与运算：
```c
#include <stdio.h>
int main(void)
{
    printf("%d\n", 3 & 2);
    return 0;
}
```
上述代码会输出2。<br />因为：
```c
00000011
00000010
//进行位与运算
00000010
=2
```
<a name="gmW3s"></a>
## (2)|
位或运算：
```c
#include <stdio.h>
int main(void)
{
	printf("%d\n", 3 | 2);
	return 0;
}
```
上述代码会输出3。<br />因为：
```c
0000 0000 0000 0011
0000 0000 0000 0010
//进行位或运算
00000011
=3
```

## (3)~
按位取反：
```c
#include <stdio.h>
int main(void)
{
    printf("%d\n", ~2);
    return 0;
}
```
上述代码会输出-7。<br />因为：
```c
0000 0000 0000 0010
//进行按位取反
1111 1111 1111 1101
=-7
```
